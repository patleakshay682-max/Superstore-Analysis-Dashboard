# 📊 Sales Analysis Dashboard

## Overview
This project presents an end-to-end Sales Data Analysis solution using Python, Pandas, Jupyter Notebook, and Power BI. It focuses on cleaning raw sales data, performing exploratory data analysis (EDA), and building an interactive dashboard to communicate meaningful business insights through effective data visualization and storytelling.

## Objectives
- Clean and preprocess sales data
- Handle missing values and duplicate records
- Perform exploratory data analysis (EDA)
- Analyze sales trends and customer behavior
- Build an interactive Power BI dashboard
- Present actionable business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI
- Microsoft Excel

## Dashboard Features
- Total Sales, Profit, Orders, and Customers KPIs
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Sales by Segment
- Top 10 Products by Sales
- Discount vs Profit Analysis
- Interactive Filters and Slicers

## Key Insights
- Technology is the highest-performing product category.
- Consumer segment contributes the largest share of sales.
- Higher discounts reduce profitability.
- Sales show an increasing trend over time.
- Regional analysis identifies top-performing markets.
